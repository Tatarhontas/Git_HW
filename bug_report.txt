1. Невозможность регистрации с паролем, содержащим только нижний регистр
   Окружение: Windows 11×64, Microsoft Edge, Версия 125.0.2535.51
   Предусловие: пользователь находится на сайте 

   Шаги:
   1. в форме Login нажать на кнопку Register 
   2. в поле Username ввести qwe
   3. в поле Password ввести asdfghjk
   4. в форме Registration нажать на кнопку Register

   Ожидаемый результат: 
   Отображается подсказка: 
   Пароль должен содержать хотя бы один символ в верхнем регистре. 

   Фактический результат:
   Отображается подсказка: 
   Успешно зарегистрировались! Пожалуйста, нажмите Назад, чтобы вернуться на страницу входа
